# Deep Learning 1 - UvA Course Assignments

This repository contains my implementations of the Deep Learning 1 course assignments from the University of Amsterdam (UvA).

## Overview

This project showcases solutions to various deep learning problems, including implementations of neural network architectures, training procedures, and theoretical analysis. The coursework covers fundamental concepts in deep learning with both practical coding exercises and theoretical components.

## Repository Structure

- **Python Scripts** - Core implementations of deep learning algorithms and models
- **Jupyter Notebooks** - Exploratory analysis and visualization of results
- **LaTeX Reports** - Detailed write-ups of solutions and theoretical discussions
- **Shell Scripts** - Utility scripts for running experiments and data processing

## Language Composition

- TeX: 67.3%
- Python: 26.4%
- Jupyter Notebook: 5.8%
- Shell: 0.5%

## Topics Covered

This repository includes assignments related to:

- Neural network fundamentals
- Backpropagation and gradient descent
- Convolutional neural networks (CNNs)
- Recurrent neural networks (RNNs)
- Graph Neural Networks (GNNs)
- Variational Autoencoders (VAEs)
- Training optimization techniques
- Regularization and dropout
- Transfer learning
- And other advanced deep learning concepts

## Setup and Requirements

### Dependencies

The project requires:
- Python 3.x
- PyTorch or TensorFlow (depending on implementations)
- NumPy
- Matplotlib
- Jupyter Notebook
- PyTorch Geometric (for GNN assignments)

Install dependencies with:
```bash
pip install -r requirements.txt
```

### Running the Code

To execute Python scripts:
```bash
python script_name.py
```

To run Jupyter notebooks:
```bash
jupyter notebook notebook_name.ipynb
```
